vue的语法学习

  • 这段代码是一个基本的 Vue 组件，它实现了一个简单的待办事项列表，用户可以添加、删除、标记完成任务，并且可以选择隐藏已完成任务。其中用到了

    ref 和 v-model 指令，用于处理表单输入和对数据的响应式更新；

    v-for 指令，用于渲染列表；

    v-bind 绑定 class 名称，用于根据状态动态设置待办事项文本的样式；

    v-if 指令，用于控制已完成任务的显示与隐藏

    v-on 指令,监听 DOM 事件，可用@代替

代码段解释：

  • (t) => t !== todo实际上就是 function(t){return t !==todo}

它的作用是从数组this.todos中筛选出所有不等于传入参数todo的元素
